Este repositório disponibiliza o código utilizado na tese “SIMULAÇÃO MULTIFÍSICA PARA EXOESQUELETOS: COMPLEXIDADE, GENERALIZAÇÃO E ESPECIFICIDADE EM AMBIENTES PERTURBADOS”, defendida no Programa de Pós-Graduação em Ciências do Movimento Humano da UDESC. O código implementa os experimentos de simulação, treinamento e avaliação dos controladores por aprendizado por reforço (TD3) descritos no trabalho.
